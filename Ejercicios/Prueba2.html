Desafio 3 del reto

Desafío: Aplicar estilos CSS sobre la página creada anteriormente para mejorar su presentación visual, sin perder funcionalidad ni accesibilidad.

Requisitos:

  1. Modificar la tipografía.
  2. Jugar con colores que mejoren el contraste.
  3. Ajustar márgenes y espaciado entre elementos.
  4. Probar posicionamientos para lograr un diseño coherente y responsive.
